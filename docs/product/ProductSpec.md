# NeonSox User Flow V1 (Conversational Onboarding)
1. Social Syncing: User connects LinkedIn, Reddit, or X.
2. Personal Greeting: App welcomes user by name with clean branding.
3. Conversational Discovery:
   - Ask for business details.
   - Ask if website exists (Yes/No toggle).
   - Dynamic branch: If no website, pivot to using connected social profiles.
4. AI Analysis State: "One Moment..." processing layout.
5. Provisioning Options: Display 3 tailored ad tier options with integrated Stripe pricing structures.
